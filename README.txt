Easily find your above the fold CSS to optimize your pagespeed.

== Description ==

My above the fold CSS let you find out your theme's - website's above the fold
CSS code (including media queries).
Enable the plugin ,Set "Enable Above The Fold CSS Display ?" to yes and visit your homepage or any other page of your website.
Thats it!Your Above The Fold CSS is bellow your footer, copy and use it as you please.

You can use the your css code with Autoptimize plugin (or any other) to boost your pagespeed rank.

= Key features =
- Generate above the fold CSS including media queries (responsive CSS).
- You can enable-disable the above the fold CSS display anytime.
- You can copy your CSS code from any page (bellow page footer).

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of My Above The Fold CSS, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “My Above The Fold CSS” and click Search Plugins. Once you’ve found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading My Above The Fold CSS plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
